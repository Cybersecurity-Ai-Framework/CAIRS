# Security Policy

CAIF is a public standards project and static website. Please report security concerns responsibly.

## Reporting a vulnerability

Open a private security advisory if the repository supports it, or contact the maintainers through the official repository contact path.

Do not disclose live exploit details, private customer data, or third-party secrets in public Issues or Pull Requests.

## Scope

Security reports may cover:

- Website or build workflow vulnerabilities.
- Unsafe documentation patterns.
- Harmful public exploit detail.
- Repository supply chain concerns.
