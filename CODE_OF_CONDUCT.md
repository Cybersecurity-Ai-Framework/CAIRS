# Code of Conduct

CAIF contributors are expected to maintain a professional, respectful, and evidence-driven community.

## Expected behavior

- Be respectful in technical disagreement.
- Keep security discussions responsible and scoped.
- Credit contributors and researchers appropriately.
- Help maintain a vendor-neutral public standard.

## Unacceptable behavior

- Harassment or personal attacks.
- Disclosure of private vulnerability details without authorization.
- Attempts to use the project to target third parties.
- Spam, astroturfing, or misleading claims.

Maintainers may moderate or remove content that violates these expectations.
