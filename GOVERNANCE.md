# Governance

The Cybersecurity AI Framework is a community-governed public standard powered by [Cyber Development](https://www.cyberdevelopment.company).

## Roles

- Members propose and discuss changes.
- Authors maintain domain-specific guidance.
- Reviewers evaluate accuracy, evidence, and consistency.
- Maintainers protect the official repository and merge accepted work.

## Decision making

Changes should be discussed publicly through Issues, Discussions, and Pull Requests. Maintainers may request revisions, additional evidence, or domain author review before merging.

## Source of truth

The official repository is the source of truth for CAIF standards, registers, mappings, scoring models, and public feeds.

## Contributor access

The public contributor model is limited to community standards work: Discussions, Issues, documentation, registers, mappings, scoring guidance, and evidence review. Private source code access, if ever needed, is handled separately by maintainers and is not granted through public contributor CTAs.
