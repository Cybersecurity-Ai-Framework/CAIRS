# Become an Author

Research domain authors help shape CAIF subject matter areas such as AI AppSec, LLM security, cloud security, API security, digital forensics, and threat intelligence.

Authors are expected to:

- Bring relevant domain experience.
- Maintain vendor-neutral guidance.
- Review community Pull Requests.
- Document evidence requirements and examples.
- Help keep the official framework consistent.

Use the "Apply to Author This Domain" issue template to request ownership or co-authorship of a domain.
