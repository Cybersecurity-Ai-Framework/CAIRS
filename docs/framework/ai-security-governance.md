# AI Security Governance

AI Security Governance defines the principles, controls, review gates, and accountability model for responsible AI use in cybersecurity validation.

## Scope

- Authorized use of AI in security testing.
- Human review requirements for AI-assisted claims.
- Traceability for prompts, tools, evidence, and decisions.
- Governance expectations for model risk, privacy, and safety.

## Contribution status

Open for community authors, reviewers, and control-mapping contributors.
