# Remediation Intelligence

Remediation Intelligence defines how CAIF prioritizes fixes, validates remediation quality, and communicates residual risk.

## Scope

- Remediation guidance quality.
- Fix validation and regression evidence.
- Complexity, ownership, and prioritization factors.
- AI-generated remediation review.

## Contribution status

Open for maintainers, remediation engineers, and application security reviewers.
