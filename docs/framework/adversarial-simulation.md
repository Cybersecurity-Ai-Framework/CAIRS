# Adversarial Simulation

Adversarial Simulation defines how CAIF maps validated weaknesses to realistic attacker behaviors, chains, and defensive outcomes.

## Scope

- Threat emulation and attack-chain validation.
- MITRE ATT&CK, OWASP, NIST, and related mapping.
- Control validation evidence.
- Simulation safety and authorization boundaries.

## Contribution status

Open for red team, detection engineering, and threat intelligence contributors.
