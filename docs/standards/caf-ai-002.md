# CAF-AI-002 Proof-of-Exploitability Evidence Standard

This draft standard defines how contributors should document proof-of-exploitability in a consistent, ethical, and reviewable way.

## Evidence expectations

- Clearly state authorization scope.
- Describe preconditions and affected components.
- Provide reproduction steps or safe proof artifacts.
- Explain exploitability without enabling unnecessary harm.
- Include remediation context where possible.

## Contribution status

This standard is open for community review and improvement.
